# ShortLinker

The backend for a simple shortlinker web app,
with a single page frontend to view the generated link.

## Technology:

* Node.js v8+
* Express.js v4
* MongoDB
* Angular4

## How to run?

1. Clone the repo.
2. npm install
3. npm start
4. To view the frontend: ng serve
